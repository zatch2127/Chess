# Chess – 2 Player Game

A simple 2-player chess game built using HTML, CSS, JavaScript, and optionally a build tool like Vite or Webpack (if applicable).

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/Chess.git
cd Chess
Install the dependencies:
npm install
Start the development server:
npm run dev
The game will run locally at http://localhost:3000 or the port specified by your dev server.
